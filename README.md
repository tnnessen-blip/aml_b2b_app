# AML Kontrollrom

Flutter app for hvitvasking/AML triage across web, Windows, macOS, Linux, iOS and Android.

## Run

```powershell
flutter pub get
flutter run -d chrome
```

For Windows desktop:

```powershell
flutter run -d windows
```

## Test

```powershell
flutter test
```
